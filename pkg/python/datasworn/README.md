# Datasworn Official Python Packages

> **⚠️ Moved.** Ported to [`datasworn-community/python-bindings`](https://github.com/datasworn-community/python-bindings) under new PyPI names (`datasworn-community-core`, `datasworn-community-classic`, `datasworn-community-delve`, `datasworn-community-lodestar`, `datasworn-community-starforged`, `datasworn-community-sundered-isles`). Python import paths unchanged (`from datasworn.core.models import Ruleset`). Use the new repo — this copy is retained for historical reference only.

Official Datasworn content packages for Python/Pydantic.

## Packages

- `datasworn-core` - Pydantic models for all Datasworn types
- `datasworn-classic` - Ironsworn Classic ruleset
- `datasworn-delve` - Ironsworn: Delve expansion
- `datasworn-starforged` - Ironsworn: Starforged ruleset
- `datasworn-sundered-isles` - Sundered Isles expansion

## Installation

```bash
uv add datasworn-core datasworn-starforged
```
