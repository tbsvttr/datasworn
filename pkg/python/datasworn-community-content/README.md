# Datasworn Community Content Python Packages

Community-created content packages for Datasworn in Python/Pydantic.

## Packages

- `datasworn-community-content-ancient-wonders` - Ancient Wonders expansion
- `datasworn-community-content-fe-runners` - FE Runners expansion
- `datasworn-community-content-starsmith` - Starsmith expansion

## Installation

```bash
uv add datasworn-core datasworn-community-content-starsmith
```
